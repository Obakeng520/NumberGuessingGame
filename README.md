Number Guessing Game - Codveda Java Development Internship

Project Overview
This repository contains a Java-based Number Guessing Game developed as part of my Level 1 (Basic) task for my internship at @Codveda. The program challenges the user to guess a randomly generated number within a set number of attempts.

Features
Random Number Generation: The system selects a random target number between 1 and 10.
Limited Attempts: Players are given a maximum of 5 attempts to guess the correct number.
Dynamic Feedback: Provides real-time "Too Low!" or "Too High!" hints after each incorrect guess.
Input Validation: includes checks to ensure the user enters a valid whole number, preventing program crashes.

Technical Stack
Language: Java
Development Environment: JDK 1.8
IDE: IntelliJ IDEA
